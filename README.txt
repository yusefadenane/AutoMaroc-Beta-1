Prototype (responsive single-page) for AutoMaroc Beta 1.

Files:
- index.html : main prototype
- style.css  : styles
- app.js     : basic javascript for language/currency/city logic
- assets/AutoMaroc_logo.png : logo (generated)

How to use:
- Unzip AutoMaroc_beta1.zip
- Open index.html in a browser (Chrome/Edge/Firefox)
- The prototype uses local static images and mock data.
- Feedback is saved to browser localStorage (no server).

Notes:
- Currency rates are static mock values in app.js.
- Replace assets/*.jpg with real Moroccan photographs named: Marrakech.jpg, Casablanca.jpg, Fes.jpg, Tangier.jpg, Agadir.jpg
